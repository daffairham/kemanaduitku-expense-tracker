# Kemanaduitku

Kemanaduitku adalah sebuah aplikasi berbasis web untuk membantu para pengguna melacak pendapatan serta pengeluaran uang.

## Tech Stack
### Front-end:
- React.js
- Tailwind CSS

### Back-end:
- Node.js
- Express
- JWT
- Bcrypt

### Database:
PostGreSQL

## Fitur
- Daftar akun,
- Login/autentikasi,
- JWT protected routes,
- Menambahkan, menampilkan, menghapus, dan update data pemasukkan/pengeluaran,
- Menampilkan riwayat transaksi
- Tampilan dashboard summary

## API Features

- Menggunakan RESTful API
- Autentikasi dengan JWT
- Protected transaction routes
- Ownership validation
- Aggregation queries untuk menampilkan summary pada dashboard


## Installation

### Clone repository ini dengan command dibawah :

```bash
git clone https://github.com/daffairham/kemanaduitku-expense-tracker
```
